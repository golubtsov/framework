## Фреймворк, разработанный на паттерне MVC

### Установка проекта локально

- из файла .env.example создать файл .env
- composer i
- npx tailwindcss -i ./public/css/input.css -o ./public/css/output.css --watch - создание стилей Tailwind
- docker compose up

### Элементы:

- Model
- Request
- Response
- View
- Router
